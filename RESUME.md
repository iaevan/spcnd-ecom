## Key architecture invariants (do not break)

- Canonical schema typing comes from sqlite schema; pg/mysql cast at factory (db/src/types.ts).
- All money strings at DB edge ("12.3400"), Money integer minor units in engines; stock as JS
  number (3dp); timestamps ISO-8601 UTC strings everywhere.
- One bus pipeline: compat-wc registers WC hook names via `bus.alias(wcName, canonicalName)`;
  never a second registry. Sync getter filters use `applyFiltersSync` (throws on async handler).
- plugin-system must NOT import core (core imports plugin-system).
- Impl packages (auth/tax/shipping/payments/email/reviews/analytics) import core interfaces
  only, never each other; register themselves in the container as plugins.
- api may import core + impls + compat-wc; apps import packages; nothing imports apps.
- Turbo `test` depends on `^build` — run `pnpm build` before cross-package tests.
